# 🛠️ Source Control Practice Repo

Welcome to the hands-on Git session! Use this repository to practice your basic workflow. Follow the ultra-summarized task checklist below.

---

## 🚀 Quick Task Checklist

### ⚙️ Step 1: Setup & Clone
- [ ] Configure identity: `git config --global user.name "Your Name"` and `user.email "email@example.com"`
- [ ] Clone this repository: `git clone <REPO_URL>`
- [ ] Move into the folder: `cd <REPO_NAME>`

### 🌿 Step 2: Branch & Work
- [ ] Create your own branch: `git checkout -b feature/your-name`
- [ ] Modify an existing file (e.g., `README.md`)
- [ ] Check status: `git status`

### 💾 Step 3: Stage & Commit
- [ ] Stage your changes: `git add .`
- [ ] Commit your changes: `git commit -m "Fix: Update documentation"`
- [ ] Create a **brand new file** (e.g., `test.txt`)
- [ ] Stage and commit the new file: `git add test.txt` then `git commit -m "Feat: Add test file"`

### ☁️ Step 4: Push & PR
- [ ] Push your branch online: `git push -u origin feature/your-name`
- [ ] Go to Bitbucket/GitHub and open a **Pull Request (PR)** from your branch to `main`

---
*💡 Hint: Stuck or messed up a local commit? Use `git reset HEAD~1` to undo your last commit while keeping your code safe!*
